# Grasshopper Quality Analyzer 🔍

> Automated quality analysis for Grasshopper definitions

**Status**: 🚧 Alpha Development | **License**: MIT | **Python**: 3.8+

---

## 🎯 Mission

Improve AEC industry productivity by bringing software engineering best practices to computational design workflows.

## 💡 Why This Matters

Computational designers spend **2+ hours** on code reviews, with **40+ questions/week** about unclear definitions. There's no quality standard, no automated checking, no best practices.

**This tool changes that.**

## 📊 Proven Concept

Pilot project at Steinberg Hart:
- **Quality**: 42/100 → 82/100 (+95%)
- **Review time**: 2 hours → 30 min (-75%)  
- **Team friction**: 40 issues/week → 5/week (-87%)

[Read the case study →](docs/case-studies/) *(coming soon)*

## ✨ Features

- **Zero setup** - Works in Grasshopper immediately
- **Real-time analysis** - See quality score as you work
- **15 lint rules** - Industry-specific checks
- **Actionable advice** - Clear fix recommendations
